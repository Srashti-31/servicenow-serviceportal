What is Service Portal?
A collection of pages that serves as the visual layer for ServiceNow, offering users an engaging way to access services, applications, tasks, and information.
Who are Service Portal and UI Builder designed for?
Service Portal is for requestors (end-users), while UI Builder is for fulfillers (support staff).
What is a widget in Service Portal?
A reusable, self-contained component that can be added to a page multiple times and used across different pages to perform searches, process requests, display forms, etc.
What framework are Service Portal pages built upon?
Bootstrap, which provides a responsive 12-column grid system that works on every device.
What can be customized using the Branding Editor?
Titles, logos, taglines, background images, and theme elements (navigation bar, page branding, text).
What are the key features of AI Search in Service Portal?
Google-like search experience, machine learning-based relevancy, and actionable top results.
What happens when you add a widget to a page?
A widget instance record is automatically created, containing configuration values specific to that particular use of the widget.
What technologies does Service Portal use?
ServiceNow API, AngularJS, Bootstrap, JavaScript, CSS+Sass (SCSS), HTML, and Font Awesome.
What accessibility standard does Service Portal comply with?
WCAG 2.1 Level AA
What is the primary navigation method in the Employee Service Center?
The Mega Menu, which provides a two-dimensional view of content hierarchy based on taxonomy.
